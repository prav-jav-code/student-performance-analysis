# 🎓 Student Performance Analysis

## 📌 Overview
This project focuses on analyzing student performance data to gain insights into academic outcomes. It uses data analysis and visualization techniques to understand how various factors affect student scores and performance.

The goal is to help educators and institutions make data-driven decisions to improve student success.

---

## 🚀 Features
- 📊 Data cleaning and preprocessing
- 📈 Exploratory Data Analysis (EDA)
- 📉 Visualization of student performance
- 🧠 Insights into factors affecting scores
- 📋 Statistical analysis of datasets

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## ⚙️ Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/prav-jav-code/student-performance-analysis.git
cd student-performance-analysis
````

2. Create virtual environment:

```bash
python -m venv venv
```

3. Activate environment:

* Windows:

```bash
venv\Scripts\activate
```

* Mac/Linux:

```bash
source venv/bin/activate
```

4. Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the Jupyter Notebook:

```bash
jupyter notebook

Open the notebook and execute cells step by step to see analysis and visualizations.
